# Ecogai Backend

This folder contains the backend API and services for the Ecogai application.

## Structure

```
backend/
├── src/
│   ├── api/          # API endpoints
│   ├── services/     # Business logic
│   ├── models/       # Data models
│   ├── utils/        # Utility functions
│   └── config/       # Configuration files
├── tests/            # Test files
└── README.md
```

## Tech Stack (Recommended)

- **AWS Lambda** - Serverless functions
- **AWS API Gateway** - REST/GraphQL API
- **AWS Cognito** - Authentication
- **AWS DynamoDB** - Database
- **AWS S3** - File storage
- **AWS Bedrock** - AI/ML services
- **Node.js/TypeScript** - Runtime

## Getting Started

1. Set up AWS account and credentials
2. Install AWS CLI and configure
3. Install serverless framework or SAM CLI
4. Deploy infrastructure
